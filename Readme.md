
# Aplicación de Chat con WebSocket
Aplicación sencilla de chat para la implementación de WebSockets

## Tecnología
* Node JS
* Express JS
* Handlebars
* SweetAlerts

## Funcionamiento
```bash
npm install 
```
```bash
npm start
```
